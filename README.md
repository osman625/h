# Mi Perfil - Proyecto Web

Este proyecto es una página web personal desarrollada como parte de una actividad académica. Incluye información sobre mí, mi experiencia, educación y un formulario de contacto funcional.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript (validación y simulación de envío POST)
